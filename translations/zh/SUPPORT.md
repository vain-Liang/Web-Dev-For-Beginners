# 支持

## 如何提交问题并获取帮助  

此项目使用 GitHub Issues 来跟踪错误和功能请求。在提交新问题之前，请先搜索现有问题以避免重复。对于新问题，请将您的错误或功能请求提交为一个新的 Issue。

关于使用此项目的帮助和问题，请参考[我们的贡献指南](CONTRIBUTING.md)。
